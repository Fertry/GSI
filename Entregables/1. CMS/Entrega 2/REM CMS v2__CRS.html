1 Introducción

Este proyecto se realiza para la asignatura de Gestión de Sistemas de Información durante el curso académico 2021/2022.

El proyecto consistirá en la realización de una página web mediante Wordpress para la empresa Salute Media, dedicada a la investigación de las enfermedades, y al desarrollo de las herramientas software necesarias para su tratamiento.
